#### 콜라츠 추측
```python3
def solution(num):
    count  = 0
    while num != 1:
        if num % 2 == 0:
            num = num // 2
            count += 1
        else:
            num = num * 3 + 1
            count += 1
        if count == 500:
            return -1
            break
            
    return count
```

#### 제일 작은 수 제거하기
```python3
def solution(arr):
    if len(arr)==1:
        arr = [-1]
    else: 
        arr.remove(min(arr))
    return arr
```
