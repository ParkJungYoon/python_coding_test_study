## 슬라이딩 윈도우(Sliding Window)

> [관련 영상](https://youtu.be/uH9VJRIpIDY)

### 알고리즘 설명

- 창문을 왼쪽부터 시작하여 한 칸씩 오른쪽으로 이동
- 매 순간 창문 안에서의 정보 유출 필요

![](./img/Sliding_Window_1.png)

- 기본 아이디어

```
Window를 한 칸 옮기면 (W-1) 칸은 겹친다!
```
